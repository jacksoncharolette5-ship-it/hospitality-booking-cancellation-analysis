# Hospitality booking and cancellationanalysis

## Overview
This project analyzes hospitality booking data to identify cancellation trends and operational impacts. The goal was to uncover patterns affecting revenue and forecasting and to provide data-driven recommendations.

## Business Problem
High cancellation rates can lead to revenue loss and staffing inefficiencies. This analysis explores key drivers of cancellations and seasonal trends to support better decision-making.

## Tools & Technologies
- Relational database design
- Tableau Public (data visualization and dashboards)
- Excel (data cleaning, pivottables)

## Data Preparation
- Cleaned and standardized raw booking data
- Designed relational tables to organize booking information
- Validated data accuracy prior to analysis

## Analysis & Insights
- Identified higher cancellation rates for bookings made further in advance
- Observed seasonal spikes in cancellations
- Detected variation in cancellation behavior across booking channels

## Recommendations
- Adjust forecasting models to account for cancellation lead time
- Use historical trends to improve staffing and inventory planning
- Implement targeted follow-ups for high-risk bookings
